HLF GOODS-N Photometric Catalog

A deep, HST-only catalog in the GOODS-N field

Oct 26, 2022

Using the aperpy code (in development) we are building a robust photometric catalog from Hubble Legacy Field GOODS-N imaging. The catalog will contain psf-matched aperture photometry from 11 HST bands from the UV to near-IR. Detailed information on the catalog creation process can be found for the HLF GOODS-S Photometric Catalog.
